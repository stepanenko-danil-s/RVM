Gwarancja / Serwis — RVM
Przyczyna w Środku maszyny → G
Z zewnątrz (brud / tara / woda / uderzenie / zużycie / czas) → NG
Внутренняя причина — гарантия; внешняя / износ / вмешательство — сервис.
Przykłady:
✅ GWARANCJA (G) — awaria wewnętrzna
  • Płyta / zasilacz (плата/блок)
  • Silnik sortera / kompaktora (двигатель)
  • Czytnik EAN / kamera (внутр. отказ)
  • Czujniki, OMRON, B3 (зав. дефект)
  • Wada fabryczna / montażu (брак завода)
❌ SERWIS (NG) — przyczyna zewnętrzna
  • Ciała obce / zła tara (постороннее/тара)
  • Zużycie: paski, rolki, szczotki (износ)
  • Brud na optyce / czujnikach (грязь → чистка)
  • Zalanie, korozja (залив)
  • Przepięcie (скачок сети)
  • Uszkodzenie / wandalizm (мех. поврежд.)
  • Brak konserwacji (нет обслуж.)
  • Samowolna naprawa (самовольный ремонт)
  • Po gwarancji (после срока)
